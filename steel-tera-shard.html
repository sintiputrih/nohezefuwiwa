A Steel Tera Shard (Japanese: テラピースはがね Tera Piece Steel) is a type of item introduced in Generation IX. It is a type of Tera Shard, which can be used to change the Tera Type of a Pokémon. It corresponds to the Steel type.

In the core series games

SVN/A$500

Effect

50 Steel Tera Shards can be used to change a Pokémon's Tera Type to Steel at the Treasure Eatery in Medali, except for Ogerpon and Terapagos.
